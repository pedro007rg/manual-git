# Manual básico de comandos Git

Este manual resume los comandos más importantes de Git para crear, guardar y compartir proyectos.

---

**git init:** Inicializa un nuevo repositorio local.  
`git init`

**git clone:** Clona un repositorio remoto en tu computadora.  
`git clone <url-del-repositorio>`

**git add:** Agrega archivos al área de preparación.  
`git add <archivo>`

**git commit:** Guarda los cambios con un mensaje.  
`git commit -m "mensaje del commit"`

**git log:** Muestra el historial de commits.  
`git log`

**git checkout:** Permite cambiar de rama o versión.  
`git checkout <nombre-rama>`

**git branch:** Crea o muestra las ramas existentes.  
`git branch`

**git push:** Envía los cambios al repositorio remoto.  
`git push origin <rama>`

**git pull:** Descarga y actualiza los cambios del repositorio remoto.  
`git pull`

**git merge:** Fusiona una rama con la actual.  
`git merge <nombre-rama>`
